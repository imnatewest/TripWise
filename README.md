# TripWise 

A full-stack AI-powered travel planning platform that simplifies the process of searching, planning, and booking travel. TripWise consolidates flights, accommodations, activities, and community travel content into one seamless experience (with the option to generate a personalized itinerary using AI.)

> Won 1st place in a UI/UX competition among 100 students, evaluated on usability, features, and design.

---

## Features

- **AI-Generated Itinerary** — Generate personalized travel itineraries, budgets, and location recommendations powered by OpenAI
- **Flight Search** — Live flight tracking and search via Flightradar24
- **Accommodations & Activities** — Browse and plan hotels and activities at your destination
- **Interactive Maps** — Real-time location and travel data via Google Maps API
- **Community Content** — Featured travel articles and itineraries shared by other users
- **Secure Authentication** — User accounts with secure login and trip data management

## Getting Started

### Prerequisites

- Ruby (latest stable version)
- Node.js & npm
- PostgreSQL
- API keys for OpenAI, Google Maps, and Flightradar24

### Running Locally

This project is not currently deployed. To run locally you'll need API keys for OpenAI, Google Maps, and Flightradar24, along with Ruby, Node.js, and PostgreSQL. Feel free to reach out if you'd like setup details.

## Project Status

This project was built for a university UI/UX design competition and is not currently deployed. All features are fully functional and can be run locally.
